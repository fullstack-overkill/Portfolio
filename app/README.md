# app

## Abra o terminal na pasta do app

## Instale as dependencias
```
npm install
```

### Inicie o servidor
```
npm run serve
```

### Compilar o html
```
npm run build
```

### Rode os testes
```
npm run test:unit
```

### Lints and fixes files
```
npm run lint
```
